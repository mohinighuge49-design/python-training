📘 About College Smart Portal

Smart Solution for College Information Management

🎯 Project Overview

College Smart Portal is a web-based application developed using Python, Flask, Bootstrap and Jinja2. This portal helps students and staff access college information quickly and efficiently.

The system provides student records, notices, college information and statistics through a modern and user-friendly interface.

🚀 Project Objectives

  • Provide digital access to college information.
  • Manage student records efficiently.
  • Display latest notices and announcements.
  • Improve communication within the college.
  • Create a modern and responsive web portal.
